Program structure

Programming: 45cr.

The program designed as a compact, students immediately dive into programing starting from the first week of the program. With this module the focus is to help the participants to build strong coding know-how and skill. The module explore the following areas of the program:

  • HTML5 and CSS3
  • Programming JavaScript
  • Programming PHP
  • Unit Testing and Quality Assurance
  • Databases and Data Structures
  • Continuous Integrations and AWS
Working as a Software Developer, 45 cr

This module about making students creating projects that based on the real-world work environment. The coding skills will be tested here. In addition to the projects students will learn about user interface design and prototyping. The module will cover the following subjects:

  • Working in a Software Development Team
  • UI Design and prototyping
  • Software Development Team Project 1
  • Software Development Team Project 2
Software implementation using framework, 30 cr

In this module students will learn advanced tools that are relevant to manage databases, version control and management of website content.

  • Content Management Systems
  • Laravel
  • React Basics
  • React Advanced
On-the-job learning

Finaly after a yearlong intensive practical learning, the student will be exposed to the real work environment. Paid or unpaid internship programs are part of the program, and it will take five months. This will enable the students to sharpen their web development skills and learn more. In the mean while it is the opportunity to establish a network and if possible to secure an employment.
