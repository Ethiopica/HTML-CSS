Projects

Pancake 1

This project is focused on creating an engaging and user-friendly online platform for selling delicious, freshly made pancakes. The page will feature a visually appealing layout with high-quality images of pancakes, a simple navigation system for easy ordering, and a secure checkout process. Users will be able to choose from a variety of pancake options, including custom toppings and extras. The page will be optimized for both desktop and mobile devices to ensure a smooth shopping experience.

Pancake 2

To enhance the user experience, we incorporated LocalStorage and JavaScript objects to store and manage user-selected choices. This allows customers to customize their pancake order by selecting different toppings and extras. The page retains these choices even after refreshing the browser. The use of LocalStorage ensures that users can easily view and modify their selections before finalizing the order and get summary of it.

Pancake 3

The third part of the pancake project is a page that collect all the orders that comes from the customers. With the help of the local storage this page will fetch every orders made by customers and will display it for the kitchen staffs. It also shows the status of each order as PENDING, READY and DELIVERED. Inadition to the statuses there is a cancel button that allow the chef to cancel orders.
